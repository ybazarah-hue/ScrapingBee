# 🔎 متتبّع ترتيب الموقع في جوجل (يومي وتلقائي)

أداة تشتغل **كل صباح تلقائياً في السحابة** (حتى لو لابتوبك مقفل وغير مشبوك بالنت)، تبحث عن كلماتك المفتاحية في جوجل، تحدد ترتيب موقعك (الصفحة + المركز)، تلتقط **سكرين شوت فعلي** لكل نتيجة، وتكتب كل شي في **Google Sheet** — كل يوم في تبويب (tab) منفصل.

التشغيل عبر **GitHub Actions** (مجاني) + **ScrapingBee** (يجيب الترتيب والصورة بمفتاح واحد).

---

## ⚙️ كيف تشتغل باختصار
1. GitHub Actions يشغّل المهمة كل يوم الساعة 6 صباحاً (الرياض).
2. السكربت يسأل ScrapingBee عن كل كلمة، صفحة صفحة، لين يلقى موقعك.
3. يلتقط سكرين شوت للصفحة اللي ظهر فيها موقعك.
4. يكتب جدول منسّق في Google Sheet + يحفظ الصور.

---

## 📋 المتطلبات (مرة وحدة)
- حساب **GitHub**
- حساب **ScrapingBee** (يعطيك 1000 كريدت مجاني للتجربة — بدون بطاقة)
- مشروع **Google Cloud** + **Service Account** (مجاني)
- ملف **Google Sheet** فاضي

---

## 🚀 خطوات الإعداد

### 1) ارفع المشروع على GitHub
- أنشئ Repository جديد، وارفع فيه كل هذي الملفات.
- **خلّه Public** عشان معاينة الصور تظهر داخل الشيت مباشرة (راجع قسم الخصوصية تحت).
  مفاتيح الـ API ما تنحفظ داخل الكود — تنحفظ مشفّرة في GitHub Secrets — فالريبو العام آمن.

### 2) مفتاح ScrapingBee
- سجّل في scrapingbee.com → من الـ Dashboard انسخ **API Key**.

### 3) Google Sheet + Service Account
1. روح `console.cloud.google.com` → أنشئ مشروع جديد.
2. من **APIs & Services → Library** فعّل **Google Sheets API**.
3. **APIs & Services → Credentials → Create Credentials → Service Account** → أنشئه.
4. ادخل على الـ Service Account → تبويب **Keys → Add Key → Create new key → JSON** → ينزّل ملف JSON. احتفظ فيه.
5. افتح ملف الـ JSON وانسخ قيمة **`client_email`** (شكلها: `...@...iam.gserviceaccount.com`).
6. أنشئ **Google Sheet** جديد فاضي.
7. خذ **معرّف الشيت (ID)** من الرابط — الجزء بين `/d/` و `/edit`:
   `https://docs.google.com/spreadsheets/d/`**`هذا_هو_الـID`**`/edit`
8. اضغط **مشاركة (Share)** على الشيت، وحط إيميل الـ Service Account (الـ `client_email`) بصلاحية **Editor**.

### 4) أضف الأسرار في GitHub
من الريبو: **Settings → Secrets and variables → Actions → New repository secret**، وأضف:

| اسم السر | القيمة |
|---|---|
| `SCRAPINGBEE_API_KEY` | مفتاح ScrapingBee |
| `GOOGLE_SERVICE_ACCOUNT_JSON` | الصق **محتوى ملف JSON كامل** |
| `SHEET_ID` | معرّف الشيت |
| `KEYWORDS_JSON` | *(اختياري)* الكلمات كقائمة JSON بدل `config.yaml` |
| `TARGET_DOMAIN` | *(اختياري)* الدومين بدل `config.yaml` |

### 5) عدّل إعداداتك
افتح `config.yaml` وحط دومينك وكلماتك ودولة/لغة البحث وعدد الصفحات.

### 6) جرّبها يدوياً
من الريبو: **Actions → Daily SERP Rank Tracker → Run workflow**.
بعد ما يخلص، افتح Google Sheet ولازم تلقى تبويب جديد باسم تاريخ اليوم وفيه النتائج + الصور.

### 7) خلّها تلقائية
خلاص — تشتغل كل يوم 6 صباحاً (الرياض). لتغيير الوقت عدّل سطر `cron` في
`.github/workflows/daily-rank.yml`.

---

## 🕒 ضبط الوقت (cron)
الصيغة `'دقيقة ساعة * * *'` **بتوقيت UTC**. الرياض = UTC+3، يعني اطرح 3 ساعات:

| الوقت اللي تبيه (الرياض) | قيمة cron |
|---|---|
| 6:00 صباحاً | `'0 3 * * *'` |
| 7:00 صباحاً | `'0 4 * * *'` |
| 8:30 صباحاً | `'30 5 * * *'` |
| 9:00 صباحاً | `'0 6 * * *'` |

---

## 💵 التكلفة
- ScrapingBee: **1000 كريدت مجاني** للتجربة.
- كل كلمة يومياً ≈ (عدد الصفحات لين يلقى موقعك) نداء بحث + 1 سكرين شوت.
- لو الكلمات كثيرة وتبي توفّر: قلّل `max_pages` في `config.yaml`.
- راقب استهلاكك من لوحة ScrapingBee.

---

## 🔐 الخصوصية
- لتظهر **معاينة الصور داخل الشيت** لازم الريبو يكون **Public** (المعاينة تعتمد على رابط الصورة من GitHub).
- لو تبي الريبو **Private**: كل شي بيشتغل عادي، بس المعاينة المصغّرة جوّه الخلية ما راح تظهر؛ تفتح الصورة من عمود **«رابط الصورة»** فقط (يحتاج صلاحية على الريبو).
- مفاتيحك دايماً مشفّرة في Secrets ولا تظهر في الكود.

---

## ⚠️ ملاحظات وقيود
- **تأخير بسيط**: أحياناً GitHub يأخّر الجدولة بضع دقائق — طبيعي.
- **بقاء الجدول شغّال**: لو الريبو خمل 60 يوم، GitHub يوقف الجدولة؛ الأداة تحفظ الصور بـ commit يومي فتبقى نشطة تلقائياً.
- **تغيّر جوجل**: ألغى جوجل `num=100` (سبتمبر 2025)، فنفحص صفحة صفحة (10 نتائج لكل صفحة) — وهذا منطق الأداة أصلاً.
- **التخصيص في الموقع (Personalization)**: نطلب النتائج بدون تسجيل دخول ومن دولة محددة عشان نقرّبها من الترتيب «المحايد».
